Very simple and customisable contact form with database integration and control panel.

== Description ==

Any Form plugin enables you to collect data from your site visitors and manage it inside your admin dashboard. 
The form has very simple configuration which is not requires any programing skils.
Email account is not required.

= Translations =

You can [translate Any Form] on [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/anyform).

== Installation ==

1. Upload the entire `any-form` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Contacts List** menu in your WordPress admin screen.
